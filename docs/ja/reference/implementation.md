## 技術詳細

### Cloudflare Workers

```typescript
import { Hono } from 'hono'

const app = new Hono()

app.get('/.well-known/did.json', (c) => {
  return c.json({
    "@context": ["https://www.w3.org/ns/did/v1"],
    "id": `did:web:${c.req.header('host')}`,
    "service": [{
      "id": "#bsky_fg",
      "type": "BskyFeedGenerator",
      "serviceEndpoint": `https://${c.req.header('host')}`
    }]
  })
})

export default app
```
